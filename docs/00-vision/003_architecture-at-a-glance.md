# Architecture at a glance

Conceptual modules:
    1. Runtime engine (scheduler + message routing)
    2. Agent interface (roles, lifecycle)
    3. Tool registry + adapters
    4. Memory (ephemeral + persistent + vector)
    5. Policy engine (pre-flight checks + approvals)
    6. Observability (traces, metrics, logs)
    7. Evaluation harness

Diagrams live in `visuals/diagrams/`.
